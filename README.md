## Getting Started

### Prerequisites
- Node.js >= 20
- npm
- Android Studio (for Android development)
- Xcode (for iOS development)

### Installation

```bash
npm install
```

### Running the App

#### Start Metro Bundler
```bash
npm start
```

#### Run on Android
```bash
npm run android
```

#### Run on iOS
```bash
npm run ios
```
